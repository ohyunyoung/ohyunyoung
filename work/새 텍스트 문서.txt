        <h1>흘러온 역사, 겹쳐지는 삶</h1>

        <p>장소: 부산도서관 2층 전시실
        <p>주최/주관: 부산도서관
        <p>참여 작가: 김유리, 박지원
        <p>참여 기획자: 오윤영
        <p>촬영 및 영상: 박천현


<p> &nbsp;&nbsp; 부산도서관 개관 5주년을 맞아 기획된 <b>《흘러온 역사, 겹쳐지는 삶》</b>은 지역 청년 작가들의 작품을 통해 도시와 개인의 기억을 돌아보는 전시이다. 전시는 부산 땅 위를 살아온 사람들의 흔적에서 출발한다. 기록되지 않은 삶은 오래된 건물이나 그림 속에 작은 흔적으로 남아있다. 김유리와 박지원 두 작가는 그 흔적 속에서 이야기를 발견했다. 이번 전시에서는 부산근현대역사관에서 선보였던 작품과 함께 두 작가의 작업 세계를 소개한다. 

        <br>
<p> &nbsp;&nbsp; 사상구의 사(沙)는 모래를, 덕포동의 덕(德)과 포(浦)는 큰 갯가 포구를 뜻한다. 지명에는 낙동강이 남해로 흘러 모래가 쌓여 만들어진 땅, 언덕 끝에 배를 대던 포구의 흔적이 남아있다. 산과 모래, 강과 바다가 만나는 부산 땅에는 오랜 기억이 스며있다. 사람들은 이곳에 집을 짓고 물고기를 잡거나, 밭을 일구며 살았다. 바다를 건너 다른 나라와 교류하기도 하고, 침략에 맞서 땅을 지키기도 했다. 일제강점기의 상처와 광복의 기쁨, 한국전쟁의 비극을 거치며 부산은 전국에서 모여든 피난민들의 터전이자 우리나라의 임시수도가 되기도 했다. 부산 곳곳에는 사람들의 희로애락이 깃들어 있다. 이 기억들은 빠른 도시 개발과 뒤섞여 오늘로 흘러왔다. 역사는 강물처럼 굽이쳐왔고, 기억은 물과 함께 흐르고 쌓였다. 전시는 두 작가의 작품을 통해 역사 속 흐름과 모래알같이 작은 개인의 이야기를 짚어본다.


<p> &nbsp;&nbsp;두 작가는 부산근현대역사관에서 피란 수도 부산의 유산을 조사하고, 각자의 시선으로 건물에 담긴 기억을 되살렸다. 김유리는 세계기상기구 선정 100년 관측소이자 지정기념물 제51호인 ‘국립중앙관상대(현 부산기상관측소)’를 주제로 작업했다. 박지원은 등록문화재 제41호 ‘임시수도 정부청사(현 동아대학교 석당박물관)’와 그 안의 유물을 새롭게 그려냈다. 김유리가 공간에 남은 삶의 흔적을 선인장으로 재구성했다면, 박지원은 유물 속 생활상을 고양이의 모습으로 재해석하여 사람들의 삶을 담았다.


        <p>&nbsp;&nbsp;<b>김유리</b>에게 공간은 시대와 삶의 흔적이 쌓이는 곳이다. 일상의 자취와 취향은 공간 속에 남는다. 선인장은 터전이 바뀌거나 환경이 척박해도 뿌리를 내리고 살아간다. 작가는 선인장을 사람의 삶에 비유한다. 김유리의 전시 공간은 선인장과 현대식 건물 구조 그림을 따라 내부로 들어가도록 구성하였다. 입구는 현재로, 안쪽으로 갈수록 과거와 현재가 겹치는 공간이 된다. 안쪽에는 ‘국립중앙관상대’를 주제로 한 작품들이 있다. 국립중앙관상대는 일제강점기인 1933년에 배 형태로 지어졌다. 피란 수도 시절에는 하루 24회의 관측으로 군사 활동과 피란민의 생업에 큰 도움을 주었으며, 지금도 부산의 기상관측소로 중요한 역할을 하고 있다. &lt;선장실의 창&gt;은 과거의 흔적이 남은 4층 창문과 창밖의 원도심 풍경을 함께 그렸다. &lt;부산기상관측소 단면&gt;은 흐르는 시간을 선인장으로 표현해 과거와 현재가 이어지는 모습을 보여준다. 종이에 펜으로 그린 &lt;지상일기도 1951-1953&gt;은 당시 기상 기록을 참고해 등압선을 선인장 형태로 이어 그린 작품이다. 강한 생명력을 가진 선인장은 시대를 이겨낸 사람들의 모습이자, 오늘을 살아가는 사람들의 모습과도 닮아있다.
</p>

        <p>&nbsp;&nbsp;<b>박지원</b>에게 고양이는 과거와 현재를 잇는 매개이자, 자유롭고 지혜로운 삶의 자세를 일깨워주는 존재다. 작가는 역사 기록과 전통 유물을 고양이의 모습으로 재해석해 화면에 담았다. 박지원의 전시 공간은 통신사의 여정을 따라 &lt;여행의 시작&gt;에서 &lt;여행길&gt;로 흐른다. 통신사는 한양(현 서울)을 떠나 동래(현 부산)에서 제사를 지낸 후, 바다를 건너 에도(현 도쿄)에 도달했다. &lt;형명기&gt; 깃발을 선두로 300에서 600명에 달하는 인원이 동행했던 이 성대한 여정 속에서, 말을 타고 기예를 펼치던 &lt;마상제&gt; 선발자들은 조선과 일본 각지에서 공연을 펼쳤다. &lt;임시정부청사_석당박물관&gt;은 일제강점기 경상도청으로, 피란 수도 시절에는 임시수도 정부청사로 사용한 건물을 그렸다. 지금은 동아대학교 석당박물관으로 운영되며, 부산 기억을 고스란히 품은 피란 수도의 중요한 유산이다. 작가는 한 화면에 여러 시간대를 담아내는 ‘동도이시법’을 활용해, 건물에 얽힌 사람들의 이야기를 그려냈다.

</p>

        <p>&nbsp;&nbsp;<b> 《흘러온 역사, 겹쳐지는 삶》은 개인의 삶과 도시의 기억, 수많은 시간이 겹치는 자리다. 두 작가는 흘러온 기억을 그림에 담았고, 그림은 보는 이의 시선과 해석 속에서 다른 기억으로 이어진다. 과거의 삶들이 여기에 와있듯, 지금 시대의 이야기도 흔적으로 남아 또 다른 기억이 될 것이다.
</p>

<script>
document.addEventListener('DOMContentLoaded', function () {
  const images = Array.from(document.querySelectorAll('.detail-images img'));
  const lightbox = document.getElementById('lightbox');
  const lightboxImg = document.getElementById('lightbox-img');
  const prevBtn = document.querySelector('.lb-prev');
  const nextBtn = document.querySelector('.lb-next');

  let currentIndex = 0;

  function openLightbox(index) {
    currentIndex = index;
    lightboxImg.src = images[currentIndex].src;
    lightbox.style.display = 'flex';
  }

  images.forEach((img, index) => {
    img.style.cursor = 'zoom-in';
    img.addEventListener('click', () => {
      openLightbox(index);
    });
  });

  prevBtn.addEventListener('click', (e) => {
    e.stopPropagation();
    currentIndex = (currentIndex - 1 + images.length) % images.length;
    lightboxImg.src = images[currentIndex].src;
  });

  nextBtn.addEventListener('click', (e) => {
    e.stopPropagation();
    currentIndex = (currentIndex + 1) % images.length;
    lightboxImg.src = images[currentIndex].src;
  });

  lightbox.addEventListener('click', () => {
    lightbox.style.display = 'none';
  });
});
</script>
